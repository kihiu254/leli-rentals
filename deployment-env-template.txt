# Instructions:
# 1. Replace all placeholder values with your actual Firebase configuration

# 3. Add these variables to your deployment platform:
#    - Vercel: Project Settings > Environment Variables
#    - Render: Service Settings > Environment
# 4. Make sure to enable for Production, Preview, and Development environments
